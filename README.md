# AI-Student-Support-Chatbot
AI-Powered Student Support Chatbot for Higher Education 
MSIT 5910 Capstone Project
University of People
Nithin Reddy

## Description

This project is a simple AI-powered chatbot developed using Python and Flask.
The chatbot answers frequently asked questions related to university admissions, registration, tuition, financial aid, and other student services.

## Technologies Used

- Python
- Flask
- HTML
- CSS
- JSON

# Phase 2 – CI/CD Enhancements

## Overview
Phase 2 introduces new features and improvements to enhance the functionality, usability, and maintainability of the AI Student Support Chatbot through an iterative development approach.

## Updates Implemented

- Added user authentication (Login/Logout) using Flask sessions.
- Restricted chatbot access to authenticated users.
- Implemented session-based chat history.
- Added **Clear Chat** functionality.
- Enhanced chatbot with greeting, help, thank-you, and goodbye responses.
- Expanded the FAQ knowledge base to include 30+ student support topics.
- Improved keyword matching and default fallback responses.
- Added automatic chat logging (`chatlog.txt`) for monitoring user queries.
- Redesigned the user interface with a modern login page, responsive chat layout, improved styling, and navigation bar.
- Refactored the application into modular components for better code organization and maintainability.
- Committed and version-controlled all enhancements through GitHub as part of the CI/CD workflow.

## Next Steps

- Integrate a SQLite database for FAQ management.
- Improve intent recognition using NLP techniques.
- Add an administrator dashboard for managing FAQs.
- Configure automated testing and deployment using GitHub Actions.
